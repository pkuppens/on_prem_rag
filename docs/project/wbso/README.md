## RVO Compliance Checklist

- [ ] **Urenadministratie**: Per project, per dag, per persoon; min. elke 10 werkdagen bijgewerkt
- [ ] **Projectadministratie**: Aard, inhoud en voortgang van S&O-werkzaamheden
- [ ] **Voortgangsrapportages**: Min. ½ A4 per kwartaal per project
- [ ] **Documenten**: Gedateerd en opsteller vermeld
- [ ] **Bewijs**: Git, issues traceerbaar
- [ ] **Bewaring**: 7 jaar
