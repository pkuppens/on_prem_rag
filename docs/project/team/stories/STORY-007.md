## User Story
As a **business user**,
I want **a web page where I can ask questions and see answers in real time**,
So that **I can quickly gain insights from my documents**.

## Acceptance Criteria
- [ ] Query box with history of previous questions
- [ ] Streaming or loading indicator while the system generates an answer
- [ ] Source citations displayed alongside each answer
