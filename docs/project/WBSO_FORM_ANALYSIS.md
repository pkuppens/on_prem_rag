## Key WBSO Requirements from Form

### 1. Project Structure Requirements

- **Project Number**: Maximum 25 characters
- **Project Title**: Maximum 200 characters
- **Project Description**: Clear description of what will be developed
- **Start Date**: Project start date in dd-mm-yyyy format
- **Phasing**: Up to 10 development phases with completion dates

### 2. Technical Requirements

- **Technical Challenges**: Must identify concrete technical problems to be solved
- **Technical Solutions**: Must describe specific technical approaches to solve challenges
- **Programming Languages**: Must specify languages, development environments, and tools
- **Technical Innovation**: Must demonstrate why solutions are technically new and challenging

### 3. Hours and Costs

- **Hours**: Planned hours for the application period
- **Costs**: Direct costs related to S&O work
- **Expenses**: Equipment and tools for S&O work
